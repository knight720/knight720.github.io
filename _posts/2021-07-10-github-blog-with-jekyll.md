## Step

1. New Repository  
Repository name: [knight720].github.io  

2. Install Jekyll  
> [Jekyll on Windows](https://jekyllrb.com/docs/installation/windows/)  

3. Create a new Jekyll site  
> [Quickstart](https://jekyllrb.com/docs/)  

4. Preview  
```
bundle exec jekyll serve
```
